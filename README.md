## Introduction
Omni-tool is a collection of mini-apps that help to replace the need for a cluttered list of applications for simple purposes. For example, omni-tool contains a fully functional calculator, including calculators for financial formulas and budgeting. A notebook, and a flash-card simulator. It also contains entertainment purposed mini-apps, such as a sound board, various game token trackers, and dice rolling and RNG.
